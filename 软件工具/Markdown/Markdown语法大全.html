原创 火星狂飙 火星狂飙 火星狂飙 2026年5月2日 22:50 北京

你是不是也有这样的经历——看到别人写的文档排版精美、结构清晰，自己却只会对着空白文档发呆？别担心，今天这篇文章，带你用一分钟掌握Markdown语法，让你的文档瞬间变得专业又好看！

Markdown是什么？简单来说，它是一种轻量级标记语言，用简单的符号就能实现加粗、标题、列表、链接、图片等各种排版效果。而且一次学会，终身受用——不管是写笔记、做文档、搭博客，Markdown都能让你事半功倍！

🔥 一、基础语法——5分钟入门

1. 标题（#）

标题是文档的骨架，用 # 来表示。一级标题最大，六级标题最小：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

效果展示：

一级标题

二级标题

三级标题

💡 小技巧：建议一篇文章只用一个一级标题，作为文章主题，其他标题层层递进使用。

2. 段落与换行

  • 段落
    ：直接回车换行即可
  • 段内换行
    ：在行末加两个空格，或使用 <br>
  • 空行
    ：两个回车，即一个空行
这是第一段文字
（空一行）
这是第二段文字

3. 字体样式

Markdown支持三种基本字体样式，通过星号 * 或下划线 _ 实现：

语法
效果
说明
*斜体*
 或 _斜体_
斜体
强调内容
**粗体**
 或 __粗体__
粗体
重点强调
***粗斜体***粗斜体
双重强调
~~删除线~~删除线
需要划掉的内容

💡 记忆口诀：一个星号是斜体，两个星号变粗体，三个星号粗斜体，加个波浪线删除线

4. 引用（>）

用 > 符号可以创建引用块，非常适合放名言、提示或重要内容：

> 这是引用内容的写法
> 可以写多行
> 每一行前面加一个 > 符号

效果展示：

这是引用内容的写法
可以写多行
每一行前面加一个 > 符号

进阶技巧：

> 一级引用
>> 二级引用
>>> 三级引用（嵌套使用）

5. 分隔线

用三个或以上的 -、*、_ 来创建分隔线：

---
***
___

效果展示：

📋 二、列表语法——让内容更有条理

1. 无序列表

使用 -、* 或 + 开头（三种符号效果相同）：

- 第一项
- 第二项
- 第三项

或：

* 苹果
* 香蕉
* 橙子

效果展示：

  • 第一项
  • 第二项
  • 第三项

2. 有序列表

使用数字加点：

1. 第一步
2. 第二步
3. 第三步

效果展示：

  1. 第一步
  2. 第二步
  3. 第三步

3. 嵌套列表

在子项前加两个空格（或一个Tab）：

1. 水果
   - 苹果
   - 香蕉
2. 蔬菜
   - 西红柿
   - 黄瓜

效果展示：

  1. 水果
    • 苹果
    • 香蕉
  2. 蔬菜
    • 西红柿
    • 黄瓜

4. 任务列表（Todo List）

这是一个非常实用的功能！

- [x] 完成文章初稿
- [x] 添加图片素材
- [ ] 校对错别字
- [ ] 排版优化

效果展示：

  •  完成文章初稿
  •  添加图片素材
  •  校对错别字
  •  排版优化

💡 小技巧：微信公众平台不支持任务列表语法，但这个语法在其他平台（如GitHub、Notion）非常有用！

🔗 三、链接与图片——图文并茂

1. 超链接

[链接文字](https://www.example.com)

示例：

想学习更多？点击 [这里](https://www.example.com) 访问官网！

效果： 想学习更多？点击 这里 访问官网！

2. 图片

![图片alt文字](图片URL)

示例：

![可爱的小猫](https://example.com/cat.jpg)

💡 小技巧：alt文字是图片加载失败时显示的替代文字，也方便搜索引擎理解图片内容。

3. 图片链接（点击图片跳转）

就是把图片语法包在链接语法里：

[![图片alt](图片URL)](跳转链接)

📊 四、代码语法——程序员的福音

1. 行内代码

用反引号 ` 包起来：

这是一段 `行内代码`，中间的内容会高亮显示

效果： 这是一段 行内代码，中间的内容会高亮显示

2. 代码块

用三个反引号 ``` 包裹，并在后面指定语言类型（可选）：

```python
def hello_world():
    print("Hello, Markdown!")
```

效果：

defhello_world():
print("Hello, Markdown!")

常用语言标识：

  • javascript
     或 js - JavaScript
  • python
     - Python
  • java
     - Java
  • html
     - HTML
  • css
     - CSS
  • sql
     - SQL
  • bash
     - 终端命令
  • json
     - JSON数据

3. 流程图代码块

```mermaid
flowchart TD
    A[开始] --> B{判断条件}
    B -->|是| C[执行操作]
    B -->|否| D[执行其他]
    C --> E[结束]
    D --> E
```

🎯 六、高级技巧——让你的文档更专业

1. 脚注

这是一段文字，需要添加解释[^1]

[^1]: 这是脚注的内容，会出现在页面底部

2. 锚点（内部链接）

## 标题一 {#title1}
## 标题二 {#title2}

[跳转到标题一](#title1)

3. 转义字符

如果你的内容本身就包含 *、# 等特殊符号，需要用反斜杠转义：

\# 这不是标题
\* 这不是列表

4. HTML支持

Markdown支持部分HTML标签，比如：

<fontcolor="red">红色文字</font>
<center>居中内容</center>
<divalign="center">居中div</div>
