告别臃肿：只需一行代码，让你的 Windows 11 像 Linux 一样丝滑

DW实践 DW实践 DW实践 2026年5月11日 17:27 美国

很多人以为 Windows 11 越用越卡，是因为电脑配置不够。

真相其实更简单。

现在的 Windows 11，本质上已经不只是一个“操作系统”。它同时还是：

  • 广告分发平台
  • 微软生态入口
  • 云同步客户端
  • 数据遥测终端
  • 后台服务集合体

你买了一台 16GB 内存的电脑，结果开机什么都不干，系统先吃掉一半。

CPU 风扇时不时狂转。 硬盘灯莫名闪烁。 开始菜单搜个文件都卡两秒。

问题不在硬件。

问题在于：系统默认替你开启了太多你根本不需要的东西。

而今天这篇文章，就是教你把 Windows 11 “减肥”。

不重装系统。 不改注册表。 不研究复杂命令。

只需要一行代码。

我们会用到一个 GitHub 上非常知名的开源工具：

Chris Titus Tech's Windows Utility

它最大的价值，不是“极限优化”。

而是：把微软藏起来的系统控制权，重新交还给用户。

为什么这类工具比“网上教程”更靠谱？

很多小白害怕“跑代码”。

一看到黑窗口，就觉得电脑快炸了。

但这里有个关键区别：

普通优化教程：

  • 到处复制注册表
  • 修改未知系统项
  • 不知道改了什么
  • 出问题无法回退

而这个工具：

  • 完全开源
  • 所有脚本都公开
  • 全球大量用户长期使用
  • 支持恢复与撤销
  • 本质是把复杂系统设置图形化

你可以理解成：

它只是帮你“集中关闭系统里的默认垃圾功能”。

并不是黑客软件。

第一步：开启工具（一行代码）

首先，右键点击桌面左下角“开始菜单”。

选择：

  • “终端（管理员）” 或者
  • “Windows PowerShell（管理员）”

这里必须强调：

一定要管理员权限。

因为它需要修改系统级设置。

然后复制下面命令。

CMD 用户

powershell -Command "iwr -useb https://christitus.com/win | iex"

PowerShell 用户

iwr -useb https://christitus.com/win | iex

按下回车后，系统会自动下载并启动工具。

第一次运行时，Windows 可能会短暂卡顿几秒，这是正常现象。

第二步：进入优化界面

工具打开后。

顶部会看到多个标签页。

点击：

Tweaks

进入后，先不要急着乱勾。

直接点击顶部的：

Standard

这是官方预设。

它会自动帮你勾选大约 80% 的基础优化项。

为什么推荐先点它？

因为很多系统服务之间存在依赖关系。

新手自己逐项研究，很容易误删关键组件。

Standard 的意义，就是：

先给你一个经过大量用户验证的“安全基础版”。

然后我们再做微调。

第三步：手动微调（核心部分）

这里才是真正决定“丝滑程度”的地方。

我把推荐配置分成两类：

  • 必选基础优化
  • 极力推荐优化

这样小白不会一次性改太多。

（不想看文字的读者可根据图片配置好后跳到第四步）

一、基础优化（Essential Tweaks）

Restore Point - Create（必须勾选）

这是整个优化里最重要的一项。

它会在修改前创建：

系统还原点

什么意思？

就是：

如果你后悔了。

或者某个软件兼容性出问题。

你可以一键回到优化前状态。

这是唯一的“后悔药”。

很多人优化翻车，不是因为改错。

而是：

没做备份。

所以这项必须开启。

Telemetry - Disable

这是：

禁用微软遥测

简单理解：

Windows 会在后台持续收集大量使用数据。

包括：

  • 使用习惯
  • 软件运行状态
  • 系统信息
  • 崩溃日志

这些东西会持续占用：

  • CPU
  • 内存
  • 网络带宽

关闭后最明显的变化：

  • 后台占用下降
  • 系统安静很多
  • 风扇转速降低

同时隐私也更干净。

属于收益极高的一项。

Widgets - Remove

就是删除任务栏那个：

“新闻 + 天气 + 推荐内容”的小组件。

这东西的问题不只是“难看”。

它本质上是一个持续联网的网页容器。

会不断刷新：

  • 新闻
  • 股票
  • 广告
  • 推荐流

对于大多数人：

完全没必要。

删除后：

  • 任务栏更清爽
  • 后台占用减少
  • 内存释放更明显

Services - Set to Manual

这是很多人优化后感觉“突然流畅”的核心原因。

Windows 默认会开机启动大量服务。

问题在于：

绝大多数人根本用不到。

比如：

  • 传真服务
  • 企业网络组件
  • 远程注册表
  • Xbox 服务
  • 打印后台监控

这些服务哪怕你不用。

它们也会：

  • 占内存
  • 抢 CPU
  • 拖慢开机

设置为 Manual 后：

系统会在“需要时”才启动。

这比直接禁用更安全。

因为：

  • 不容易误伤系统
  • 兼容性更稳定
  • 风险更低

属于非常聪明的优化策略。

Hibernation - Disable（重点分析）

这是争议最大的一项。

它的作用是：

禁用休眠功能

很多人勾完之后。

突然发现硬盘多了：

5GB 到 15GB 空间。

为什么？

因为 Windows 的休眠机制，会创建一个巨大的隐藏文件。

名字叫：

hiberfil.sys

它会把内存内容保存到硬盘。

所以内存越大，占用越大。

比如：

  • 16GB 内存
  • 可能占用 6GB 到 10GB

这项到底该不该关？

先说优势：

优势

  • 立刻释放大量磁盘空间
  • SSD 压力降低
  • 后台机制减少
  • 系统更纯净

特别适合：

  • 256GB 小硬盘用户
  • C 盘爆红用户

风险

但它并不是“零代价”。

关闭后：

  • 开机可能慢 3 到 5 秒
  • 笔记本彻底没电时，无法自动保存当前工作状态

什么意思？

以前电脑突然没电。

系统可能还能恢复你之前开的网页和文档。

关闭休眠后。

没了。

我的建议

如果你：

  • 台式机
  • SSD 足够大
  • 经常合盖休眠
  • 很在意断电恢复

可以不关闭。

如果你：

  • C 盘空间告急
  • 追求极简
  • 基本不用休眠

建议关闭。

这才是真正理性的优化。

不是“无脑全关”。

二、进阶与视觉优化（极力推荐）

Background Apps - Disable

这是：

禁止后台应用偷跑

很多 Windows 应用。

哪怕你没打开。

也会在后台持续运行。

比如：

  • 天气
  • 地图
  • Xbox
  • 微软商店

它们会：

  • 偷吃内存
  • 后台联网
  • 消耗 CPU

关闭后：

系统响应速度会明显更干净。

Microsoft OneDrive - Remove

很多人对 OneDrive 最大的不满：

不是同步。

而是：

强制同步

桌面文件。 文档。 图片。

微软总想替你“自动接管”。

结果就是：

  • 开机弹窗
  • 文件路径混乱
  • 后台上传
  • 网络占用

如果你不用微软云盘。

建议直接移除。

系统会清爽很多。

Right-Click Menu Previous Layout - Enable

这是我最推荐的一个。

开启后：

Windows 11 会恢复：

Win10 风格右键菜单

不用再点：

“显示更多选项”。

效率提升极大。

属于：

每天都能感知到的优化。

Start Menu Bing Search - OFF

关闭开始菜单的网页搜索。

这是很多人忽略的卡顿来源。

你本来只是想搜：

“微信”

结果 Windows 还要联网给你搜网页。

于是：

  • 开始菜单卡顿
  • 搜索延迟
  • CPU 突增

关闭后：

本地搜索会快很多。

体验接近 Linux 的即时启动。

Start Menu Recommendations - OFF

关闭开始菜单推荐内容。

那些：

  • 最近文件
  • 广告推荐
  • 莫名其妙的软件提示

大多数人根本不需要。

关闭后：

开始菜单终于像个工具。

不是广告位。

Settings Home Page - OFF

关掉设置页面的“首页”。 现在的设置页面充斥着微软的订阅广告和云盘提醒，关掉它，让设置回归纯粹的选项列表。

S0 Sleep Network Connectivity - OFF

（笔记本用户必看）

很多笔记本在书包里莫名其妙发烫、耗电

就是因为这个所谓的“现代待机”。

关掉它，让电脑合盖后真正进入“安静睡眠”。

第四步：锁定更新策略

接下来进入：

Updates 页面

这里是很多人最容易忽略，但实际最关键的一步。

直接点击中间最显眼的按钮：

Security Settings

这意味着：

系统只接收：

  • 安全补丁
  • 漏洞修复

而不会被微软强行推送：

  • 新功能
  • 半成品更新
  • 莫名 Bug

这是目前 Windows 最稳定的更新策略。

第五步：锁定更新策略

全部设置完成后。

点击左下角按钮：

Run Tweaks

命令行窗口会开始自动执行系统命令。

期间：

  • 不要关闭窗口
  • 不要强制重启
  • 不要乱点

等它提示：

Tweaks are Finished

就说明完成了。

第六步：重启电脑

最后一步非常关键：

因为很多系统服务。

必须重启后才会真正生效。

重启完成后。

你会明显感受到：

  • 开机更安静
  • 内存占用下降
  • 搜索响应更快
  • 系统动画更跟手
  • 后台进程减少

那种“微软在后台偷偷干活”的感觉。

会明显减弱。

最后的风险提示

任何系统优化。

本质上都是：

用“部分默认功能”，交换“系统控制权”。

所以一定记住一句话：

Restore Point 是唯一后悔药。

不要跳过。

另外。

这套方案的核心思路，并不是“魔改 Windows”。

而是：

把那些你根本不用、却长期占资源的东西关掉。

Windows 11 本身并不差。

真正让它臃肿的

是微软默认替所有人做了太多决定。
