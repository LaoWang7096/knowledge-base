Codex桌面版通过Codex plus plus工具接入 DeepSeek V4保姆级教程，目前最简单的方案

原创 阿俊520 阿俊520 科技爱好者站 2026年5月27日 19:28 广东

想要实现 Codex 接入国产模型实现额度自由吗? 

下载一个 EXE 安装程序就完事了，全程可视化、零代码、小白也能一次成功，文章中有工具安装包下载地址。


为什么要搞？

Codex 桌面版只认 OpenAI 自家模型。想用便宜的 DeepSeek V4？因为协议不兼容——Codex 走 Responses API，DeepSeek 走 Chat Completions。

以前要搞一堆代理转换，现在不用了。
Codex++ 自带协议转换，图形界面配置，一个 EXE 安装程序全部搞定。

✅ 需要的准备


  • ✓ Windows 电脑

  • ✓ 已装好 Codex 桌面版（微软商店搜 Codex）

  • ✓ DeepSeek API Key（platform.deepseek.com 注册）

💡 DeepSeek V4：输入 1 元/百万 tokens，输出 2 元/百万 tokens，充 10 块用两个月。

第一步：下载 Codex++

打开项目地址：

https://github.com/BigPizzaV3/CodexPlusPlus

点 Releases → 最新版 v1.1.7 → 下载：

CodexPlusPlus-1.1.7-windows-x64-setup.exe



下载之后，双击安装，一路下一步即可。



 第二步：配置 DeepSeek

桌面会有两个新图标：Codex++（入口）和 Codex++ 管理工具（设置）。



打开Codex++ 管理工具：


在供应商/模型配置 → 添加供应商 → OpenAI Chat 兼容 → 填入：

  • API 地址：https://api.deepseek.com
  • API Key：你的 DeepSeek Key（https://platform.deepseek.com 中注册并创建API key）

上游协议：选择：Chat Completions


如下图：


点击保存配置。


最后，启用刚配置好的模型。



第三步：启动 Codex + +

双击 Codex++ 启动。


输入指令进行测试，发现可以正常回复，并可以看到模型自动切换为deepseek v4 pro了。



目前测试，插件功能正常。比如测试生成一个英文问候的视频。


🎉 恭喜，完成了！


常见问题

Q：提示连接失败？
A：检查 Codex++ 管理工具是否在后台运行。

Q：API Key 在哪？
A：platform.deepseek.com/api_keys 创建。

Q：想恢复官方版？
A：卸载 Codex++ 即可。

总结

下载一个 EXE → 填一个 Key → 完事。 小白也能 3 分钟搞定。
